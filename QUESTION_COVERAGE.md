## üöÄ Usage Instructions

### Generate All Questions
```bash
# Run the comprehensive generation script
./scripts/run-question-generation.sh

# Or run directly with TypeScript
npx ts-node scripts/generate-comprehensive-questions.ts
```

### Question Structure
Each question includes:
- **Company Context**: Specific to the company's interview style
- **Position Relevance**: Tailored to the role requirements
- **Difficulty Progression**: Easy ‚Üí Medium ‚Üí Hard
- **Topic Tags**: For filtering and categorization
- **Seniority Levels**: Entry, Junior, Mid, Senior

### Database Organization
Questions are stored in Firestore with:
- **Collection**: `practice_questions`
- **Indexing**: By company, position, difficulty, tags
- **Metadata**: Creation date, update date, usage statistics

## üîÑ Maintenance & Updates

### Adding New Companies
1. Add company to `companies` array in generation script
2. Define company focus areas
3. Add company-specific question templates (optional)
4. Run generation script

### Adding New Positions
1. Add position to `positions` array
2. Define relevant skills and categories
3. Update `getRelevantCategories` function
4. Run generation script

### Updating Questions
1. Modify question templates in generation script
2. Update company-specific adaptations
3. Re-run generation script
4. Questions are versioned with update timestamps
